# AI Agent（练习仓库）

本仓库从本地 `LESSON_HM/ai/agent` 同步，按**顶层子目录**分类存放各 demo（每个子目录是独立小项目，**不需要**再 `git init`）。

## 目录说明

| 目录 | 说明 |
|------|------|
| `langchain_nest` | Nest + LangChain 示例 |
| `memory-demo` | 对话记忆相关实验 |
| `rag_book` | RAG / 电子书 |
| `milvus-test` | Milvus 向量库练习 |
| `mcp` / `mcp_in_action` | MCP 相关 |
| `mini-cursor` | 小 demo |
| `RAG` | RAG 拆分与加载等 |

安装依赖请在**各子目录**内单独执行 `pnpm install` / `npm install`。
